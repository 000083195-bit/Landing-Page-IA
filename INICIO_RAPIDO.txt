🚀 CÓMO COMENZAR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASO 1: CREAR BASE DE DATOS
  
  En terminal (Windows):
  $ mysql -u root < database.sql
  
  O en PHPMyAdmin:
  1. Crear base de datos "pesado_fallo"
  2. Importar archivo database.sql
  3. Verificar que se crearon las tablas

PASO 2: CONFIGURAR VARIABLES DE ENTORNO
  
  1. Copiar .env.example a .env
  2. Editar .env:
     DB_HOST=localhost
     DB_USER=root
     DB_PASS=(tu contraseña si existe)
     DB_NAME=pesado_fallo
     DB_PORT=3306

PASO 3: INICIAR SERVIDOR PHP
  
  En terminal:
  $ php -S localhost:8000
  
  Luego abrir: http://localhost:8000/login.php

PASO 4: PROBAR CON CREDENCIALES DE EJEMPLO
  
  ADMIN:
  - Email: admin@pesado.com
  - Contraseña: password123
  - Acceso: Dashboard
  
  USER:
  - Email: user@pesado.com
  - Contraseña: password123
  - Acceso: Landing Page

PASO 5: DESPLEGAR EN RAILWAY (Cuando esté listo)
  
  Ver archivo: RAILWAY_DEPLOYMENT.md
